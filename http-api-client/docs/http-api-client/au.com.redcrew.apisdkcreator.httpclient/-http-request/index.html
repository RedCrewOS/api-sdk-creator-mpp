HttpRequest

In order to be as generic as possible properties on the request try to align with the underlying structure of an HTTP request.

Adapters will have to map properties to the request object structure used by the underlying HTTP client library.

data class HttpRequest<T>(method: HttpRequestMethod, url: HttpRequestUrl, headers: Map<String, String>, pathParams: Map<String, String>?, queryParams: Map<String, String>?, body: T?)
